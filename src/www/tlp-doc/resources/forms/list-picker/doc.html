List picker offers an easier way to pick values from a <select>. It is meant to be easier to use and to customise than Select2, while having a similar user experience.
If you need an auto-completer, a way to search among a lot of values fetched asynchronously, if each group of items can be fetched separately from the others, if you need to add new values dynamically, please consider using Lazybox instead.

Please note: There are predefined templates available by using special data-attributes on <option> tags:

  • Use data-avatar-url with the value of avatar url to display user avatar. Example: data-avatar-url="https://my-avatar.example.com"
  • Use data-color-value with the value of the color to display a colored circle next to the option label. The color must be either a tlp-color (for example inca-silver) or in hexadecimal (for example #8f8f8f). Example: data-avatar-url="fiesta-red"
You cannot have both datasets on the same option.

You can also use the items_template_formatter callback to display these templates.

Options:
  • locale: string | undefined: A locale string formatted with underscores. For example: "en_US", "fr_FR". It is used to localize List Picker. If undefined, it will fallback to "en_US"
  • placeholder (default: ""): A (translated) text inviting the user to fill up the field.
  • is_filterable (default: false): A boolean to add a search input in the list picker dropdown
  • none_value: string (default: null): The none value given on the select. If not null, the option with this value will be automatically selected when no other value is selected. It will also be removed from the selection when any other value is selected.
  • items_template_formatter (default: none): A callback function returning a lit/html template result to customise items templates.
    It takes three parameters:
    • html: HTMLTemplateStringProcessor The lit/html tag function to help the developer to build a secured TemplateResult
    • option_value: string The value of the current select box <option>
    • option_label: string The text content of the current select box <option>
