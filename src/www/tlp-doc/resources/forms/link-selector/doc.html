Link selector allows users to select an artifact to link

It is basically a lite version of list-picker, but tailored for a very specific, non-generic use case.

Options:

  • locale: string | undefined: A locale string formatted with underscores. For example: "en_US", "fr_FR". It is used to localize link-selector. If undefined, it will fallback to "en_US"
  • placeholder (default: ""): A string message inviting the user to fill up the field
  • items_template_formatter (default: none): An asynchronous callback function returning a lit/html template result to customise items templates
    It takes three parameters:
    • html: HTMLTemplateStringProcessor The lit/html tag function to help the developer to build a secured TemplateResult
    • option_value: string The value of the current select box option
    • option_label: string The text content of the current select box option
  • search_field_callback: LinkSelectorSearchFieldCallback (mandatory): A callback to be executed each time the user types a query in the search input.
    This callback takes a query: string in its parameters, which contains the query typed by the user, and returns void.
    In this callback, you are expected to filter the collection of HTMLOptionElement inside the source HTMLSelectElement.
    To do so, you can:
    1. Clear the source select options
    2. Compare the options to the received query or fetch some on a distant endpoint
    3. Insert the matching options inside the source select
    4. Insert an empty state in the source select if the query did not match any item (see details below)

Empty state:

There is a special select option that will be identified and used by the link-selector as an empty state.

This option should have the following attributes and values:
  • value="" (no value). Link-selector will ignore this option as it has no value.
  • disabled to prevent it to be selected.
  • data-link-selector-role="empty-state" to mark the option as an empty state.
Please note that these empty states are only supported in optgroups.
