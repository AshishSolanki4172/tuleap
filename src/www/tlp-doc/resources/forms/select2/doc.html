Select2 is deprecated.

  1. If you need an easier way to pick values from a <select>, you have few possible values (fewer than 100), or you need <form> validation / submit, consider using List picker.
  2. If you need an auto-completer, a way to search among a lot of values fetched asynchronously, if each group of items can be fetched separately from the others, if you need to add new values dynamically, consider using Lazybox.
