• When there are both cancel and primary action buttons, then the primary button is always on the right.
  • To be able to use large buttons on the footer you should use the class tlp-modal-footer-large.
  • Buttons that have data-dismiss="modal" attribute in the modal will close it.
  • A click on the backdrop or hitting the Escape key will also close the modal.
  • Elements that have data-modal-focus attribute in the modal will take focus when it opens.
  • You must explicitly call the modal in javascript: const the_modal = createModal("element-id");. This will return a Modal instance. You can then call show(), hide(), or toggle() methods on this instance.

Arguments:

  • element: Element
  • options: ModalOptions (Optional).
    Option Type Default Description
    keyboard boolean true When true, enables closing the modal when pressing the Escape key.
    destroy_on_hide boolean false When true, destroys the event listeners when hiding the modal. Useful when for example you create new modals on a button click.
    dismiss_on_backdrop_click boolean true When true, dismiss the modal when user clicks on the backdrop.

Events:

Type When is it dispatched? Target Cancelable? Bubbles? Effect
tlp-modal-shown On modal.show() document and the Modal instance No No None
tlp-modal-will-hide After clicking on the backdrop, after clicking on a data-dismiss="modal" button, after pressing the Escape key the Modal instance Yes No If you cancel it with event.preventDefault(), the modal will NOT be hidden.
tlp-modal-hidden On modal.hide() document and the Modal instance No No None
